# 🇰🇷 2026 재정사업 분석 플랫폼

대한민국 전체 재정사업을 분석하는 비공식 오픈소스 대시보드입니다.

🔗 **라이브 데모**: `https://withakiss.github.io/budget2026`

> ⚠️ 본 플랫폼은 개인이 분석 목적으로 제작한 비공식 페이지입니다. 공식 예산 자료로 활용할 수 없습니다.

---

## ✨ 주요 기능

| 탭 | 내용 |
|---|---|
| 📊 **개요** | KPI 카드, 기능별 예산 비중, 증감 Top 10, 신규/계속 사업 분포 |
| 🏛️ **부처별 분석** | 부처별 예산 규모·증감률·포트폴리오 카드 |
| 📂 **기능별 분석** | 교육·복지·국방·과학기술 등 기능분류별 분석 |
| 📋 **사업 목록** | 검색·필터·정렬 지원 전체 사업 테이블 |
| 🔍 **중복 분석** | 키워드 기반 부처 간 유사·중복 사업 탐지 |
| ⚠️ **리스크 분석** | 이상치(±50%) 탐지, 회계유형별 구성 |

## 🔄 실제 데이터로 교체

`data/projects.json` 파일의 구조를 유지하면서 내용만 교체 가능

### 데이터 소스

1. **열린재정** (openfiscaldata.go.kr) → 세출예산현황 Excel
2. **공공데이터포털** (data.go.kr) → "세부사업별 예산" 검색
3. **기획재정부** (moef.go.kr) → 예산안 첨부파일

### JSON 필드 설명

```json
{
  "id": "P001",                    // 고유 ID
  "ministry": "과학기술정보통신부",   // 부처명
  "function": "과학기술",           // 기능분류
  "project_name": "사업명",         // 사업명
  "project_type": "R&D",          // R&D / 정보화 / 일반
  "is_new": false,                 // 신규여부
  "budget_2025": 115000,          // 2025 예산 (백만원)
  "budget_2026": 145000,          // 2026 예산 (백만원)
  "change_amount": 30000,         // 증감액
  "change_rate": 26.09,           // 증감률(%)
  "account_type": "일반회계",       // 회계유형
  "description": "사업 설명",       // 설명
  "sub_projects": [               // 내역사업 (선택)
    { "name": "세부사업명", "budget": 60000 }
  ]
}
```
