Jaccard|A∩B| / |A∪B|집합 기반, 단어 유무만 판단, 빠름짧은 사업명 비교
CosineA·B / (‖A‖×‖B‖)벡터 각도, 단어 빈도 반영설명문 비교
Dice2|A∩B| / (|A|+|B|)작은 집합에 유리, Jaccard 변형짧은 텍스트
Overlap|A∩B| / min(|A|,|B|)부분집합 관계 탐지에 강함포함 관계 탐지
TF-IDF Cosinetf × log(N/df) → cosine희귀 단어에 높은 가중치 부여의미적 키워드 검색
BM25 ★IDF × tf(k₁+1) / (tf + k₁(1−b+b·dl/avgdl))문서 길이 정규화 + 포화 처리. TF-IDF보다 정밀정보검색, 긴 설명문 매칭
Hybrid0.5 × Jaccard + 0.5 × Cosine집합·벡터 방식의 균형 조합빠른 일반 탐색
다차원 가중합 ★Σ(차원 × 프로필 가중치)7차원 동시 분석, 프로필별 맞춤 조정 가능정밀 중복 탐지
